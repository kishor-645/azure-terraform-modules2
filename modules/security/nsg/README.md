# Network Security Group (NSG) Terraform Module

Dynamic NSG with service tags, CIDR ranges, and Application Security Groups support.

## Features
- ✅ Dynamic inbound/outbound rules
- ✅ Service Tags support
- ✅ Application Security Groups
- ✅ Multiple subnet associations
- ✅ Log Analytics integration

## Requirements
- Terraform >= 1.10.3
- azurerm ~> 4.51.0
