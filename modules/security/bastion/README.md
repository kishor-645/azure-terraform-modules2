# Azure Bastion Terraform Module

Azure Bastion Standard with native client, IP-based connections, and shareable links.

## Features
- ✅ Standard SKU
- ✅ Native Client support
- ✅ IP-Based Connections
- ✅ File Copy and Copy/Paste
- ✅ Zone Redundancy

## Requirements
- Terraform >= 1.10.3
- azurerm ~> 4.51.0
