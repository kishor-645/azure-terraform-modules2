# Route Table (UDR) Terraform Module

User-defined routes for traffic steering through Azure Firewall or NVAs.

## Features
- ✅ 5 next hop types
- ✅ Azure Firewall integration
- ✅ BGP route propagation control
- ✅ Multiple subnet associations

## Requirements
- Terraform >= 1.10.3
- azurerm ~> 4.51.0

v1.0.0 (November 2025)
