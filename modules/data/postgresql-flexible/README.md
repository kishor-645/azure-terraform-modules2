# PostgreSQL Flexible Server Module

PostgreSQL v16 with HA, geo-redundant backup, and private endpoint.

## Features
- PostgreSQL v16
- GP_Standard_D4s_v3 (4 vCPU, 16 GB RAM)
- 128 GB storage (expandable)
- Zone-redundant HA
- Geo-redundant backup (7-day retention)
- Private endpoint support
- Custom configuration parameters

## Requirements
- Terraform >= 1.10.3
- azurerm ~> 4.51.0
