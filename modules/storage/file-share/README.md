# Azure File Share Terraform Module

Premium Azure File Share with NFS 4.1 protocol support.

## Features
- ✅ Premium tier (TransactionOptimized)
- ✅ NFS 4.1 protocol
- ✅ Private endpoint
- ✅ 100 GB default quota
- ✅ Diagnostic logging

## Requirements
- Terraform >= 1.10.3
- azurerm ~> 4.51.0
