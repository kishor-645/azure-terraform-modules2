# Azure Container Registry Terraform Module

Premium ACR with private endpoint and geo-replication support.

## Features
- ✅ Premium SKU
- ✅ Private endpoint
- ✅ Geo-replication
- ✅ Admin account disabled
- ✅ Customer-managed encryption
- ✅ Retention and trust policies

## Requirements
- Terraform >= 1.10.3
- azurerm ~> 4.51.0
