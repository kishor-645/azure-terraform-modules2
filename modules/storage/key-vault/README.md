# Azure Key Vault Terraform Module

Key Vault with RBAC authorization and private endpoint.

## Features
- ✅ Standard/Premium SKU
- ✅ RBAC authorization (recommended)
- ✅ Purge protection enabled
- ✅ Private endpoint
- ✅ Network ACLs
- ✅ Audit logging

## Requirements
- Terraform >= 1.10.3
- azurerm ~> 4.51.0
