# Storage Account Terraform Module

Azure Storage Account with private endpoint support and ADLS Gen2.

## Features
- ✅ Standard/Premium tiers
- ✅ Private endpoint for blob and file
- ✅ ADLS Gen2 (hierarchical namespace)
- ✅ Customer-managed encryption keys
- ✅ Blob versioning and soft delete
- ✅ Network ACLs

## Requirements
- Terraform >= 1.10.3
- azurerm ~> 4.51.0
