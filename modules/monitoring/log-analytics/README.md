# Log Analytics Workspace Module

Centralized logging workspace with Container Insights and Security solutions.

## Features
- PerGB2018 SKU
- 30-day retention (configurable)
- Container Insights for AKS
- Security solution
- Capacity reservation support

## Requirements
- Terraform >= 1.10.3
- azurerm ~> 4.51.0

v1.0.0 (November 2025)
