# Application Insights Module

Workspace-based Application Insights for application telemetry.

## Features
- Workspace-based (integrated with Log Analytics)
- Connection string output
- 90-day retention
- 10 GB daily cap
- Support for web, Java, Node.js applications

## Requirements
- Terraform >= 1.10.3
- azurerm ~> 4.51.0

v1.0.0 (November 2025)
