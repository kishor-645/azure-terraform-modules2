# Troubleshooting Guide

Common issues and solutions.

## Terraform Issues

- Backend initialization fails: Check storage account permissions
- State lock timeout: Break lease with caution

## AKS Issues

- Cannot connect: Get credentials from jumpbox
- Nodes not ready: Check node events

## Networking Issues

- Pods cannot reach internet: Verify UDR to firewall
- Cannot access private endpoints: Check DNS resolution
